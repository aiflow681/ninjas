【游戏名称】忍者突袭

【游戏介绍】
这是一款H5原生开发的反应类小游戏。游戏中，你扮演一名忍者，使用手裏剑进行攻击和躲避。敌人会从屏幕上方下来，你要保持警惕，消灭敌人保护自己！

【玩法说明】
操作：
- 使用屏幕右侧的"攻击"按钮向最近的敌人投掷手裏剑
- 使用屏幕左侧的"躲避"按钮进行短暂闪避
- 游戏会根据屏幕大小自动调整控制按钮

道具：
- 手裏剑：每击败一个敌人会得到分数
- 忍者：击败它可获得额外分数
- 武士：击败它可获得大量分数

目标：
- 击败尽可能多的敌人，获得高分
- 当敌人到达屏幕底部时，你会失去生命值
- 生命值归零时游戏结束

Tips：
- 游戏结束会显示你的最高分数记录
- 连续击败敌人可以增加连击数，获得额外分数
- 合理使用攻击和躲避按钮是游戏成功的关键

【技术介绍】
HTML5原生开发，主要语言是JavaScript，使用Canvas进行游戏渲染，并添加了移动端优化代码。
